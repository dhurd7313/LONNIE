LONNIE LOCAL AGENT
==================

This gives the LONNIE web app real desktop control:
  - Run terminal commands (PowerShell)
  - Take screenshots of your entire screen (including Docker Desktop)
  - Manage Docker containers
  - Read/write files anywhere on your machine
  - Read/write clipboard
  - Interactive WebSocket terminal

HOW TO USE:
1. Double-click START-LONNIE-AGENT.ps1
   (or right-click > Run with PowerShell)

2. Copy the Auth Token shown in the terminal window

3. In LONNIE web app > Settings (gear icon) > Agent section
   - Enter: http://localhost:45678
   - Paste the auth token
   - Click Connect

4. LONNIE will now have full desktop access

KEEP THE AGENT WINDOW OPEN while using LONNIE.

REQUIREMENTS:
- Node.js (nodejs.org)
- Docker (optional, for Docker features)
